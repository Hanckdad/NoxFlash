# Hatsune Flash - AI Assistant

Advanced AI Assistant with powerful tools including:
- Unrestricted AI conversations using Gemini API
- File and image analysis
- TikTok, Instagram, YouTube download tools
- AI image tools (face filters, anime conversion)

## Features

### AI Chat
- Unrestricted conversations using Gemini 2.0 Flash
- Code blocks with syntax highlighting
- Markdown support
- Chat history persistence

### File Analysis
- Upload images for analysis
- Document upload (PDF, DOC, TXT, etc.)
- OCR and content analysis

### AI Tools
- **TikTok**: Download videos without watermark
- **Instagram**: Download photos and videos
- **YouTube**: MP4 video and MP3 audio download
- **AI Image Tools**: 
  - Face filters (coklat, hitam, nerd, piggy, carbon, botak)
  - Image to anime conversion
